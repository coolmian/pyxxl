    Executor

    执行器，真正的调度任务和策略都在这里

    Parameters:

    Name Type Description Default
    xxl_client XXL

    xxl客户端

    		 required
    handler Optional[JobHandler]

    Defaults to None.

    		 None
    loop Optional[asyncio.AbstractEventLoop]

    Defaults to None.

    		 None
    max_workers int

    执行同步任务的线程池. Defaults to 20.

    		 20
    task_timeout int

    全局的任务超时配置,如果executorTimeout参数存在,以executorTimeout为准. Defaults to 60*60.

    		 60 * 60
    max_queue_length int

    单机串行的队列长度，当阻塞的任务大于此值时会抛弃. Defaults to 30.

    		 30

    thread_pool = ThreadPoolExecutor(max_workers=max_workers, thread_name_prefix='pyxxl_pool') instance-attribute
