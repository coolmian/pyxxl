    PyxxlRunner

    Example 
    runner = PyxxlRunner(
    "http://localhost:8080/xxl-job-admin/api/",
    executor_name="xxl-job-executor-sample",
    port=9999,
    host="172.17.0.1",
    handler=xxl_handler,
)

    Parameters:

    Name Type Description Default
    xxl_admin_baseurl str

    xxl-admin服务端暴露的restful接口url(如http://localhost:8080/xxl-job-admin/api/)

    		 required
    executor_name str

    xxl-admin上定义的执行器名称,必须一致否则无法注册(如xxl-job-executor-sample)

    		 required
    handler JobHandler

    执行器支持的job,没有预先定义的job名称也会执行失败

    		 None
    access_token str

    xxl-admin的认证token,如果没有开启不需要传. Defaults to None.

    		 None
    host str

    执行器绑定的host,xxl-admin通过这个host来回调pyxxl执行器,如果不填会默认取第一个网卡的地址. Defaults to None.

    		 None
    port int

    执行器绑定的http服务的端口,作用同host. Defaults to 9999.

    		 9999

    access_token = access_token instance-attribute

    handler = handler or JobHandler() instance-attribute

    host = ensure_host(host) instance-attribute

    port = port instance-attribute
